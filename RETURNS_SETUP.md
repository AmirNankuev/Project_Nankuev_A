# Возвраты

Изменения добавляют пользовательское оформление возврата из личного кабинета.

## Что добавлено

- Кнопка «Оформить возврат» у оплаченных заказов в личном кабинете.
- Страница создания заявки на возврат.
- Страница просмотра статуса заявки.
- Поле фото товара в заявке на возврат.
- Обработка возвратов в Django Admin: одобрить, отклонить, завершить.
- При завершении возврата товар в состоянии «Новый» автоматически возвращается на склад.

## После копирования файлов

Выполнить:

```bash
python manage.py migrate
python manage.py runserver
```

## Проверка

1. Оформить заказ.
2. Выполнить тестовую оплату.
3. Открыть личный кабинет.
4. Нажать «Оформить возврат» у товара в заказе.
5. Заполнить причину и состояние товара.
6. Проверить заявку в админке: Main → Возвраты.
