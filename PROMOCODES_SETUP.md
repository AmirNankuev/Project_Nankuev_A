# Промокоды и скидки

Добавлен модуль промокодов для учебного интернет-магазина одежды.

## Что появилось

- модель `PromoCode` в приложении `main`;
- поле `promo_code` у заказа;
- поле `discount_amount` у заказа;
- управление промокодами в Django Admin;
- ввод промокода в корзине и на первом шаге оформления заказа;
- пересчёт итоговой суммы с учётом скидки;
- сохранение применённого промокода в заказе;
- увеличение счётчика использований промокода после создания заказа;
- отображение скидки на страницах корзины, checkout, оплаты, личного кабинета и панели менеджера.

## Как применить изменения

После копирования файлов выполните миграции:

```bash
python manage.py migrate
```

## Как создать промокод

1. Войдите в Django Admin под администратором.
2. Откройте раздел `Main → Промокоды`.
3. Создайте промокод, например:
   - код: `SALE10`;
   - тип скидки: `Процент`;
   - размер скидки: `10`;
   - минимальная сумма заказа: `1000`;
   - активен: да.
4. Сохраните.

## Типы скидок

### Процент

Например, `SALE10` со значением `10` даст скидку 10% от суммы товаров.
Можно указать максимальную скидку, чтобы ограничить размер скидки.

### Фиксированная сумма

Например, `NEW500` со значением `500` даст скидку 500 ₽.
Если сумма товаров меньше скидки, итоговая стоимость товаров не уйдёт ниже 0 ₽.

## Ограничения

Промокод можно ограничить:

- датой начала действия;
- датой окончания действия;
- минимальной суммой заказа;
- максимальным количеством использований;
- флагом активности.

## Где пользователь вводит промокод

- в корзине;
- на первом шаге оформления заказа.

На шаге доставки промокод уже только отображается. Чтобы изменить его, пользователь возвращается на шаг данных или в корзину.
